Top 3 Programming Language to Learn in 2020!

Kisan Tamang ,   15 Feb 2020  3 min read

The most prestigious skill you can develop in today's digital world is coding. It teaches you how to think and how to solve problems.

Coding teaches you to understand the foundation of the computer system. How a computer achieves its task and how it works.

There are tons of programming languages in the market. They have their own capabilities. You might be confused about which language you should choose as a beginner.

Don't worry in this article I am going to talk about the top programming language to learn in 2020 and how they can be used to achieve your dream projects.

Here is my list of top 3 programming language to learn in 2020

1. Python

Python is an interpreted, high level, general-purpose programming language. It can use for multiple purposes. It is easy to learn and beginner-friendly.

Here is a simple python code: 


         # a function to find even or odd number
         def evenOdd(num):
            if num % 2 == 0:
              print(f"{num} is even")
            else:
              print(f"{num} is odd")

Python is extremely used for Machine Learning and professional developer also uses it to build web, mobile, and desktop application as well.

But Python is not suitable for desktop and mobile applications. I will talk about a suitable language shortly. It is much popular in backend development, machine learning, data science, automation, and scripting.

According to the StackOverflow Developer Survey 2019, Python is the most wanted programming language in 2019.

Python can be used in multiple fields like Data Science, Web Development, Machine Learning, Scripting, Software Development, Automation, etc.

According to indeed.com, the average salary of python developers in the United States is $120k +.

Python being my personal favorite language, I would definitely suggest you get started with Python in 2020.

2. JavaScript

JavaScript is high level, interpreted or just-in-time compiled programming language with first-class functions. Ok, you might be thinking what the hell he is talking about. right?

Well in simple language, JavaScript is a programming language of the web. More than 75% of websites on the internet use JavaScript.

Here is a simple snippet of JS: 


              //function to find the even or odd number
              function evenOdd(num){
                  if(num % 2 == 0){
                    return "Even"; 
                  }else{
                    return "Odd";
                  }
                }

According to the StackOverflow Developer Survey report of 2019, JavaScript is the second most wanted language.

JavaScript has tons of amazing libraries and framework which make front-end development rapid. Here is the StackOverflow survey report of 2019 showing the top Web Frameworks.

JavaScript can be used to build web, mobile, and even desktop applications. Most used javascript libraries are React, Vue.js, jQuery, Angular, etc.

3. Java

Java is a Compiled programming language used in enterprise application development. It is used by plenty of software companies around the world to build large scale applications.

Java is also used to build native android applications. If you want to become an app developer, Java is right for you. It can also be used to develop websites.

Here is Java Code to find Even or Odd number:


          import java.util.Scanner;

            class OddOrEven
            {
              public static void main(String args[])
              {
                int x;
                System.out.println("Enter an integer to check if it's odd or even");
                Scanner in = new Scanner(System.in);
                x = in.nextInt();

                if (x % 2 == 0)
                  System.out.println("The number is even.");
                else
                  System.out.println("The number is odd.");
              }
            }

Of course, it looks tedious than Python and JavaScript code mentioned earlier, but Java is a great language. There are lots of books about data structure, system design, software architect written in Java. So knowing Java is just like knowing English.

At last, my suggestion is that get started with one of the above languages. When you become comfortable with one language, it is much easier to catch the next one.

Good Luck!!
